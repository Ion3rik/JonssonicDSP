Detailed Description

template<typename T, typename Interpolator = LinearInterpolator<T>>
class jnsc::DelayLine< T, Interpolator >

A multichannel delay line class for audio processing with fractional delay support.

Parameters
TSample data type (e.g., float, double)
InterpolatorInterpolator type for fractional delay support (default: LinearInterpolator)

Process a block of samples for all channels with modulated delay.

Parameters
inputInput sample pointers (one per channel)
outputOutput sample pointers (one per channel)
modulationModulation signal pointers (one per channel, in samples)
numSamplesNumber of samples to process
Note
The modulation signal is added to the base delay time. Modulation values should be in samples and will be clamped to valid range [0, bufferSize-1].
Input, output, and modulation must all have the same number of channels as prepared.
