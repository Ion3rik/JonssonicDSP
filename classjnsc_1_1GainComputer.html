Detailed Description

template<typename T, typename Policy = CompressorPolicy<T>>
class jnsc::GainComputer< T, Policy >

GainComputer class template. Computes gain based on input signal level and specified dynamics policy.

Template Parameters
TSample type
PolicyDynamics policy. (Default: CompressorPolicy<T>, see note)
Note
Current policies include CompressorPolicy, ExpanderDownPolicy, ExpanderUpPolicy, LimiterPolicy, and GatePolicy. Policy classes are defined in detail/gain_computer_policies.h.
