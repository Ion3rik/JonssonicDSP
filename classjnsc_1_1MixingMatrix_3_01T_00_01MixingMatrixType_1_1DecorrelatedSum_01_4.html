Detailed Description

template<typename T>
class jnsc::MixingMatrix< T, MixingMatrixType::DecorrelatedSum >

Mixing matrix that implements a decorrelated sum mixing strategy. Each output channel is a normalized sum of all input channels with alternating signs.

Note
Uses the Dense mixing matrix internally.
