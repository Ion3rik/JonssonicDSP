Detailed Description

template<typename T, size_t Factor = 4>
class jnsc::Oversampler< T, Factor >

Oversampler class for upsampling and downsampling audio signals. Additionally integrated latency compensation (e.g. for alligning dry/wet signals).

Template Parameters
TSample data type (e.g., float, double)
FactorOversampling factor (supported: 2, 4, 8, 16)
Note
Currently Fixed FIR halfband filters are used for each stage.
