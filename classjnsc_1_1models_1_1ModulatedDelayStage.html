Detailed Description

template<typename T, typename Interpolator = LinearInterpolator<T>, bool UseInternalLFO = true, bool UseDamping = false, bool UseCrossFeedback = false>
class jnsc::models::ModulatedDelayStage< T, Interpolator, UseInternalLFO, UseDamping, UseCrossFeedback >

Modulated delay stage with LFO modulation.

Template Parameters
TSample data type (e.g., float, double)
InterpolatorInterpolator type for delay line (default: LinearInterpolator)
UseInternalLFOIf true, includes an internal LFO for modulation.
UseDampingIf true, includes a one-pole lowpass damping filter in the feedback path.
UseCrossFeedbackIf true, enables cross-feedback between channels for ping-pong effects.
