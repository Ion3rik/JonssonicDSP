Detailed Description

template<typename T, WaveShaperType ShaperType, bool PreFilter = true, bool PostFilter = true, size_t OversamplingFactor = 1>
class jnsc::models::SaturationStage< T, ShaperType, PreFilter, PostFilter, OversamplingFactor >

Saturation stage model with oversampled waveshaping, pre- and post-filters, and pre and post dynamics.

Template Parameters
TSample data type (e.g., float, double)
ShaperTypeType of wave shaper nonlinearity
PreFilterIf true, enables pre-waveshaper filtering
PostFilterIf true, enables post-waveshaper filtering
OversamplingFactorOversampling factor (1, 2, 4, 8, or 16)
