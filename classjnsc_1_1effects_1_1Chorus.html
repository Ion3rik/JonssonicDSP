Detailed Description

template<typename T>
class jnsc::effects::Chorus< T >

Chorus audio effect.

Classic chorus effect using a modulated delay line with feedback. Creates sweeping comb-filter effects by mixing the input signal with a time-varying delayed copy of itself.
