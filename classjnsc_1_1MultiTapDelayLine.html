Detailed Description

template<typename T, size_t NumTaps, typename Interpolator = detail::LinearInterpolator<T>>
class jnsc::MultiTapDelayLine< T, NumTaps, Interpolator >

A multichannel delay line class for audio processing with fractional delay support.

Parameters
TSample data type (e.g., float, double).
NumTapsNumber of delay taps.
InterpolatorInterpolator type for fractional delay support (default: LinearInterpolator).
