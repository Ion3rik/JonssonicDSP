Detailed Description

template<typename T>
class jnsc::effects::Flanger< T >

Flanger audio effect.

Classic flanger effect using a short modulated delay line with feedback. Creates sweeping comb-filter effects by mixing the input signal with a time-varying delayed copy of itself.
