Detailed Description

template<typename T, size_t Factor>
class jnsc::OversampledProcessor< T, Factor >

OversampledProcessor class template. Compile-time and runtime-switchable oversampling processor wrappers.

Compile time version of OversampledProcessor with fixed oversampling factor. This class uses a single Oversampler instance configured at compile time.

Template Parameters
TSample data type (e.g., float, double)
FactorOversampling factor (supported factors: 2, 4, 8, 16)
Note
If Factor is 0, runtime-switchable specialization is used.
Template Parameters
TSample data type (e.g., float, double)
FactorOversampling factor (supported factors: 2, 4, 8, 16)
Note
This class is more efficient when the oversampling factor is known at compile time.
