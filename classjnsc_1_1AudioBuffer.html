Detailed Description

template<typename T, BufferLayout Layout = BufferLayout::Planar>
class jnsc::AudioBuffer< T, Layout >

A multi-channel audio buffer supporting both planar and interleaved layouts.

Layout is selected via the BufferLayout template parameter:

  • Planar: [Ch0_Sample0, Ch0_Sample1, ..., Ch1_Sample0, Ch1_Sample1, ...] (Each channel's samples are stored contiguously for cache-friendly per-channel access.)
  • Interleaved: [Sample0_Ch0, Sample0_Ch1, ..., Sample1_Ch0, Sample1_Ch1, ...] (All channels for each sample are stored contiguously for cache-friendly per-sample access.)
Template Parameters
TSample type (typically float or double)
BufferLayoutLayout of the buffer (default: Planar)
Note
All the Jonssonic DSP processors use planar layout internally for simplicity and performance.
